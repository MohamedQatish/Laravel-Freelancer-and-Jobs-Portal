# Laravel Jobs & Freelancing Portal  

## Overview  
This project is a comprehensive jobs and freelancing platform built using Laravel. It offers a seamless experience for both job seekers and employers with advanced features like:  
- User authentication and profile management.  
- Job posting and application tracking.  
- Email notifications for job applications and status updates.  
- Rating system to evaluate freelancers and employers.  
- Contracts and wallet integration for payment management.  
- Skill-based search and favorite jobs management.  

With clean code standards and efficient database management powered by Laravel's Eloquent ORM, the platform ensures performance and scalability.

---

## Key Features  
- **User Authentication:** Secure login, registration, and role-based access.  
- **Profile Management:** Customizable user profiles for freelancers and employers.  
- **Job Management:** Create, edit, and manage job postings with advanced filters.  
- **Dynamic Relationships:** Polymorphic models and morphable relationships for data flexibility.  
- **Rating System:** Review and rate freelancers and employers.  
- **Wallet Integration:** Manage payments securely and efficiently.  
- **Skill Management:** Search and filter by skills or specialization.  
- **Favorite Jobs:** Save and manage favorite job postings.
